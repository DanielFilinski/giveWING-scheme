# Молитвенная комната — Материалы и реквизит

**Формат:** интерактивная молитвенная тропа  
**Вместимость:** 1–8 человек за заход  
**Время на группу:** 15–20 минут

---

## Помещение

- Отдельная **палатка** (или комната)
- Управление светом — возможность полного отключения
- Минимум 2 стола или стула для расстановки реквизита
- Телефон со скачанной музыкой

---

## Декор и атмосфера

- [ ] Светодиодные свечи — много (есть)
- [ ] Фонарь / лампа для подсветки креста (с возможностью вкл/выкл)
- [ ] Ткань однотонная для декора мебели
- [ ] Небольшой кусочек красной ткани — для креста (есть)
- [ ] Цветы для декорации
- [ ] Гирлянда

---

## Реквизит по станциям

### Станция 1 — Сотворение

- [ ] Картина / распечатка с изображением природы: животные, птицы, красивые пейзажи
- [ ] Гирлянда
- [ ] Воздушные шарики белые — **24 шт** (2+2 на заход × 6 заходов = 1 пачка)

### Станция 2 — Грехопадение

- [ ] 10 заповедей — распечатать (или церковный настенный календарь)
- [ ] Ветка + змея (доделать змею!)
- [ ] Надкусанное яблоко + игла (взять свою)

### Станция 3 — Смерть Христа

- [ ] Свеча настоящая + подсвечник + спички (взять свои)
- [ ] Игла (взять свою)
- [ ] Шарик с водой
- [ ] Шарик с надписью «Иисус Христос»
- [ ] Изделие из гвоздей и дерева (брусок с гвоздями) — есть
- [ ] Миска для лопания шарика с водой (взять на кухне)
- [ ] Крест бумажный — есть
- [ ] Колючий венок — есть

### Станция 4 — Праведность Христа

- [ ] Пенопласт — **1 лист 100×100 см** (нарезать на квадраты 10×10 см), закупить в строительном магазине
- [ ] Ацетон — **2 л на 50 человек**, закупить в строительном магазине
- [ ] Посуда для ацетона (потом выкинуть)
- [ ] Поднос для пенопласта (взять на кухне?)
- [ ] Маркер чёрный — **1 шт** — есть
- [ ] Бутылка с водой (найти на месте)
- [ ] Распечатать: **«Праведность Христа»** (документ)
- [ ] Распечатать: **сценарий**

---

## Команда ведущих

**Минимум 2 человека:**

| Роль | Задача |
|---|---|
| **Чтец** | Ведёт рассказ, раздаёт реквизит участникам, заводит группу, следит за помещением |
| **Мим** | Показывает физико-химические опыты под текст. Одет в **чёрную одежду** |

---

## Суть программы

Раскрывается великая борьба: Творение → Грехопадение → Смерть Христа на Голгофе → Дух Святой → Призыв к покаянию.  
В конце — время личной молитвы к Богу.
