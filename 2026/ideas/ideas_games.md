# Идеи для общелагерных игр — Слёт 2026

> Аудитория: взрослые 35+.
> Формат: командные игры на весь лагерь. Все команды участвуют одновременно.
> Принцип: азарт + юмор + лёгкая связь с темой.

---

## Игра 1. Морской бой по категориям

**Суть:** Классический морской бой, где координаты — это не буквы и цифры, а категории и значения. Команды «стреляют», называя пересечение двух категорий.

**Подходящее время:** свободный вечер или после обеда (60–90 мин)

**Количество команд:** 2–6 команд (можно играть попарно или все против ведущего)

---

### Как это устроено

**Игровое поле** — таблица: по горизонтали одна категория, по вертикали другая.

**Пример поля:**

|  | Ветхий Завет | Новый Завет | Псалтирь | Притчи | Пророки |
|---|---|---|---|---|---|
| Человек | | | | | |
| Место | | | | | |
| Событие | | | | | |
| Число / дата | | | | | |
| Имя Бога | | | | | |

**Каждая клетка = «корабль»** — правильный ответ, заранее заготовленный ведущим.

**Команда «стреляет»:** «Человек / Новый Завет» — ведущий открывает клетку. Если команда даёт верный ответ за 30 секунд — «попадание», получает очко. Если нет — «промах», клетка закрывается, ход переходит к следующей команде.

---

### Варианты полей

#### Поле А — Библейское
**По горизонтали:** Ветхий Завет, Новый Завет, Псалтирь, Притчи, Апокалипсис
**По вертикали:** Человек, Место, Событие, Число, Имя/титул

*Примеры ответов в клетках:*
- Человек / Апокалипсис → Иоанн Богослов (автор)
- Место / Ветхий Завет → Вавилон, Синай, Иерихон (на выбор)
- Число / Псалтирь → 150 (псалмов), 23-й (самый известный)

#### Поле Б — «Лагерное»
**По горизонтали:** День 1, День 2, День 3, День 4, День 5
**По вертикали:** Тема дня, Утренний страж, Вечерняя игра, Слово/стих, Спикер/ведущий

*Команды вспоминают, что было на слёте — работает в последний день.*

#### Поле В — «Христианин и мир»
**По горизонтали:** Молитва, Слово, Отношения, Работа, Испытания
**По вертикали:** Призыв из Библии, Ловушка/соблазн, Практический шаг, Пример из жизни Иисуса, Вопрос для размышления

*Более дискуссионный формат — ответы не всегда однозначны, ведущий заранее готовит «эталонные» варианты.*

---

### Правила игры

1. Ведущий заранее готовит поле — таблицу с заполненными ответами (спрятаны от команд).
2. Команды ходят по очереди, называя координату (категория + значение).
3. Ведущий открывает клетку и озвучивает задание/вопрос.
4. У команды **30 секунд** на ответ (без подсказок от других команд).
5. Правильный ответ = 1 очко + право сделать ещё один ход (как «попадание» в морском бое).
6. Неправильный ответ = ход переходит к следующей команде.
7. Победитель — команда с наибольшим количеством очков по итогу поля.

**Усложнение:** у некоторых клеток — «линкор» (3 очка, но за неправильный ответ теряете 1 очко).
**Упрощение:** командам можно совещаться 60 секунд вместо 30.

---

### Что нужно подготовить

- [ ] Таблица-поле (распечатать крупно или вывести на экран/флипчарт)
- [ ] Закрытые ответы для каждой клетки (листок у ведущего)
- [ ] Таймер (30 / 60 сек)
- [ ] Счётная доска или маркерная доска с очками команд
- [ ] Маркеры / стикеры для отметки «подбитых» клеток
- [ ] Колокольчик или сигнал конца времени

**Подготовка:** 30–40 мин на составление поля и проверку ответов.

---

### Связь с темой слёта

Поле Б и В напрямую связаны с темой «Христианин последнего времени». Игра в конце слёта (День 6–7) по полю Б — живое повторение всего, что было, через азарт командного соревнования.
