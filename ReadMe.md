# GiveWing — Молодёжный слёт 2026

## О проекте

Разработка программы молодёжного слёта 2026 года на основе анализа слёта «Дайвинг 2025».

**Целевая аудитория:** молодые люди 30+,христиане, незамужние и несемейные, весёлые и простые, ценящие живое общение, юмор и глубину одновременно.

---

## Правила работы над проектом

### 1. Контекст и источники

- Все решения принимаются с опорой на материалы из `references/` — реальные программы, расписания и сценарии слёта 2025
- Перед предложением нового — сначала изучить, что уже работало. Не изобретать велосипед там, где есть проверенное
- Файл `references/2025.md` — главный источник знаний о прошлом слёте
- `references/расписание/schedule.md` — структура и логика расписания

### 2. Аудитория 30+: ключевые принципы

Эти люди отличаются от молодёжи 18–25. Учитывай:

- **Ценят глубину**, но без занудства — смешно и серьёзно одновременно
- **Не любят детские игры** — форматы должны быть «взрослыми» по духу, но игривыми по исполнению
- **Ценят время** — никакого «резинового» расписания, чёткие тайминги
- **Много рефлексии** — готовы говорить о настоящих вопросах: отношения, призвание, страхи, вера
- **Физически активны**, но без фанатизма — спорт как радость, не как нагрузка
- **Социальные** — нетворкинг, знакомства, живое общение важнее формальных активностей
- **Усталые от «церковного формата»** — ценят живость, нестандарт, искренность

### 3. Баланс программы (обязателен)

Каждый день должен включать все четыре блока:

| Блок | Назначение | Примеры |
|------|-----------|---------|
| **Духовный** | Рост, глубина, встреча с Богом | Утренний страж, проповедь, молитвенная тропа |
| **Активный** | Энергия, тело, приключение | Спорт, экскурсии, командные игры |
| **Рефлексивный** | Осмысление, личное | Командное общение, вопросы на еду |
| **Социальный** | Связи, радость, смех | Вечерние программы, настолки, костёр |

Нельзя перегружать один блок за счёт другого.

### 4. Структура дня (базовый шаблон)

```
07:00 — Подъём / зарядка
08:00 — Утренний страж (по командам)
08:30 — Завтрак
09:30 — Активный блок (спорт / море / выезд)
11:30 — Семинар / обучение
13:00 — Обед
14:00 — Общелагерная игра или мастер-классы
18:00 — Ужин
19:00 — Вечерняя программа (главный блок дня)
21:00 — Общение по командам
22:00 — Свободное время / настолки / костёр
```

### 5. Командная система

- Команды — основа социальной динамики слёта
- Оптимальный размер: 5–7 человек
- Команды формируются в первый вечер через интерактивное задание (как в 2025 — разрезанные стихи)
- У каждой команды: капитан, название, своя идентичность
- Командное общение — ежедневно, не пропускать

### 6. Духовный блок: правила

- **Тема слёта** — единая нить через все дни, каждый день раскрывает один аспект
- **Утренний страж** — короткий (30 мин), практичный, с вопросами для обсуждения
- **Вечернее служение** — кульминация дня, соединяет тему с личным опытом
- **Молитва** — не формальная, живая, по теме дня
- Избегать: морализаторства, давления, «правильных ответов»
- Приветствовать: честность, сомнения, живые вопросы

### 7. Форматы, доказавшие эффективность (из 2025)

- Разбивка на команды через интерактивное задание на открытии
- Бенгальские огни на командном общении (первый день)
- Солёное/несолёное печенье как иллюстрация темы
- «ПУП» — Простой, Уникальный, Полезный факт о себе
- Вопросы на приёмы пищи — для плавного вхождения в тему дня
- Экскурсии на выбор (2 раза за слёт)
- Костёр у моря / у воды
- Голосование «звёздами» на плакатах команд

### 8. Что нужно обновить для 2026

- Тема слёта — новая, не повторять «7 навыков»
- Игры и активности — обновить, добавить свежие форматы
- Вечерние программы — новые сценарии
- Утренние стражи — под новую тему
- Командные задания — новые, соответствующие аудитории 30+

### 9. Идеи для новых форматов (специфика 30+)

- **«Стол переговоров»** — командные дискуссии на острые темы с защитой позиции
- **Networking-вечер** — структурированные быстрые знакомства (скоростные свидания по интересам)
- **«Открытый микрофон»** — свидетельства, стихи, песни — 3 минуты без подготовки
- **Квест по территории** с духовным смыслом
- **Кино-вечер** с обсуждением (короткометражка + разговор)
- **Мастер-классы от участников** — каждый эксперт в чём-то своём
- **«Вопросы без ответов»** — обсуждение сложных богословских вопросов без давления на «правильный ответ»
- **Творческий итог** — каждая команда создаёт что-то к закрытию (видео, постер, песня)

### 10. Тайминг вечернего служения

> Применяется ко всем вечерним служениям, **кроме понедельника** (открытие), **пятницы** (встреча субботы) и **воскресенья** (закрытие). Выступление команды — вторник, среда, четверг, суббота.

```
18:45 — Раздача почты — выступающая команда
19:00 — Ролик дня
19:05 — Гимн — ведущие приглашают всех к исполнению
19:10 — Выступление команды
19:30 — Ведущие приглашают воздать хвалу вместе с группой прославления
19:31 — Пение — группа прославления
19:40 — Ввод в тему — ведущие
19:55 — Призыв к молитве перед духовными наставлениями — ведущие
20:00 — Проповедь (40 мин) — Комарницкий
20:40 — Молитва — Комарницкий
20:50 — Общее пение — приглашают ведущие
```

### 11. Тональность и стиль

- Искренний, тёплый, живой — не «официальный церковный»
- Юмор уместен везде, кроме молитвы
- Ведущие — равные участникам, не над аудиторией
- Пространство для честности и уязвимости
- Нет давления на «правильное» поведение или эмоции

### 12. Правила подбора игр

Аудитория — незамужние и несемейные. Это определяет жёсткое ограничение по физическому контакту.

**Недопустимо:**
- Любой телесный контакт между участниками разного пола: удержание, перенос, поддержка
- Игры, где механика построена на прикосновениях (сидение на коленях, голова к голове, шар между телами)
- Эстафеты с предметами между телами (газета лбами, шар между коленями)

**Пример недопустимой игры:** «Большая эстафета» (День 2, Вариант Д) — этапы «Газета», «Стул», «Воздушный шар» содержат недопустимо-близкий контакт и **не могут использоваться** в программе.

**Допустимо:**
- Командные игры без физического контакта между участниками
- Эстафеты с предметами (ложка/мяч, конусы, хула-хуп без удержания за руки)
- Интеллектуальные и творческие форматы
- Подвижные игры с чёткой дистанцией между участниками

## Как работать с AI-ассистентом над этим проектом

1. **Начинай с контекста:** укажи, что именно разрабатываешь — день, блок, конкретный формат
2. **Ссылайся на 2025:** «как в прошлом году, но...» — ассистент знает материалы из `references/`
3. **Уточняй аудиторию:** напоминай про 30+, если нужен специфический подход
4. **Проси конкретику:** не «придумай игру», а «придумай командную игру на 60 минут для открытия, аудитория 30+, тема — доверие»
5. **Итерируй:** первый вариант — черновик, уточняй детали в диалоге
